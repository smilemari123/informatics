Вариант 3
Пользователь вводит число с плавающей точкой. Считать число и вывести отдельно целую и дробную часть. Предусмотреть возможность наличия ведущих нулей у дробной части.
Бонус: предусмотреть обработку ограничений ввода (например, если число считывается в виде строки, то завершать работу программы при превышении заданной длины).
Пример ввода: 10.05
Пример вывода: 10 | 05

Задача разделения числа с плавающей точкой решена двумя способами.
Если требуется просто соблюсти формат вывода: <число до точки>|<число после точки>, то можно использовать программу в файле simple.c.
Если важно, чтобы на вывод подавались два числа типа int, то надо выполнять файл program_3v.c.

Максимальное количество символов, учитывая точку, 63.
